# mcl

![Stability: experimental](https://img.shields.io/badge/stability-experimental-orange.svg)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![License: Apache](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)
![Github action](https://github.com/status-im/nim-mcl/workflows/CI/badge.svg)

Simple [mcl](https://github.com/herumi/mcl) wrapper for Nim, fully integrated with the Nim build system.

Applications using `nim-mcl` are fully stand-alone, needing no additional DLL or shared library.
